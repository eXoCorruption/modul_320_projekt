# modul_320_projekt
Modul 320 Projekt.


Aufbau

1.1	Öffnen
Mit einem Button den Prozess starten.

1.2	Garagentor
Das Tor öffnet sich mit Countdown (evt. 15 sec für Vorschau) fürs Schlissen.

1.3	Lichtschranke
Sensor, welcher das Fahrzeug einfährt und den Countdown resettet, fürs Schlissen.
Könnte anhand eines Buttons simuliert werden.

1.4	Beleuchtung mit Countdown
Einschalten bei Einfahrt mit Zeitbegrenzung

1.5	Protokoll
Erfassung der Einfahrtszeit mit Zeitstempel.
